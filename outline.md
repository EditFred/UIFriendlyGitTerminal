# Git Vibes
A UI friendly terminal for git actions

## Platform
- Mac OS

## API & Database
- Connect to the github account on mac

## Synopsis
This is going to be a mac os app to make git actions easier. Instead of typing out git pull, push, merge, switch etc. The UI will offer buttons to do the actions instead. A user will open the app, navigate to their git projects, and control git from there. No more terminal. There should be an option to allow the user to log in if necessary. BUT assume the user is already logged into git within their terminal.

## UI & UX Design
- A friendly design.
- Actions are made by clicking buttons. Not typing into terminal
- Support Light & Dark mode. Or match local system settings


### P0
- Allow a user to fulfill all their needed git actions.
- Pull, Merge, Switch, Push, Commit(with message)
- Easy file navigation to the git project
- Easily see branches
### P1
